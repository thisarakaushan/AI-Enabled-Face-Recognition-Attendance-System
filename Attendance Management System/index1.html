How it works?

The project functions by integrating facial recognition technology into an attendance management system through a web application. Initially, the faculty member can choose between two options: capturing live images using the webcam or uploading a class photo. Upon selection, the system employs face recognition algorithms to identify students present in the images. Subsequently, the system records the attendance by saving the date and time in a CSV file. For verification purposes, the faculty can query the attendance records by specifying the date, start time, and end time of the class session. The system then retrieves and displays the relevant attendance information, aiding the faculty in effectively managing student attendance. Overall, the project streamlines attendance tracking processes by leveraging facial recognition technology within a user-friendly web interface.
